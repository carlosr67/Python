Análisis de Normales Climatológicas de Colombia con Python
Descripción del Proyecto

Este proyecto fue desarrollado en Google Colab utilizando Python con el objetivo de analizar las Normales Climatológicas de Colombia y obtener información relevante sobre los patrones climáticos del país.

A través del procesamiento, limpieza y análisis de datos meteorológicos históricos, se identifican tendencias relacionadas con precipitación, temperatura, humedad y otras variables climáticas, permitiendo conocer las características ambientales de los diferentes departamentos y regiones de Colombia.

El análisis busca responder preguntas como:

¿En qué departamentos llueve más?
¿Cuáles son las zonas más cálidas del país?
¿Dónde se registran las temperaturas más bajas?
¿Qué regiones presentan mayores niveles de precipitación?
¿Cómo se distribuyen las variables climáticas a nivel nacional?

Objetivos del Proyecto
Analizar datos climatológicos históricos de Colombia.
Identificar patrones climáticos por departamento.
Detectar regiones con mayores niveles de precipitación.
Analizar el comportamiento de la temperatura en distintas zonas del país.
Generar visualizaciones que faciliten la interpretación de los datos.
Extraer información útil para estudios ambientales, agrícolas y territoriales.

Fuente de Datos

Los datos utilizados corresponden a registros climatológicos recopilados por entidades oficiales encargadas del monitoreo meteorológico en Colombia.

Las variables analizadas incluyen:

Temperatura media.
Temperatura máxima.
Temperatura mínima.
Precipitación acumulada.
Humedad relativa.
Altitud de estaciones.
Ubicación geográfica.
Departamento y municipio.

Proceso de Análisis

1. Extracción de Datos
Carga de archivos CSV y Excel.
Consolidación de información climática.
2. Limpieza y Transformación
Tratamiento de valores nulos.
Corrección de formatos.
Estandarización de variables.
Validación de registros inconsistentes.
3. Análisis Exploratorio (EDA)
Distribución de temperaturas.
Distribución de precipitaciones.
Identificación de valores extremos.
Comparación entre departamentos.
4. Visualización de Datos
Mapas geográficos.
Gráficos de barras.
Histogramas.
Diagramas de dispersión.
Ranking de departamentos.

Principales Análisis Realizados
Departamentos con Mayor Precipitación

Identificación de las regiones que presentan los mayores acumulados de lluvia durante los periodos analizados.

Departamentos Más Cálidos

Análisis de las temperaturas medias y máximas para determinar las zonas con mayor sensación térmica.

Departamentos Más Fríos

Comparación de temperaturas mínimas para identificar regiones de clima frío y de alta montaña.

Distribución Climática Nacional

Visualización geográfica de variables climáticas para comprender mejor las diferencias entre regiones.

Tendencias Climáticas

Identificación de comportamientos y patrones relevantes dentro de los datos históricos.
